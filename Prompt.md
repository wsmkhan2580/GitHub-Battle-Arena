GitHub Battle Arena

## Prompt 1 — Project Setup
"Create a new React project using Vite.
Set up the folder structure with components,
services, and utils folders inside src."

---

## Prompt 2 — GitHub API Integration
"Write a api.js file in services folder.
It should fetch user data from GitHub API
using async await and Fetch API.
No Axios. Handle 404 and network errors."

---

## Prompt 3 — Search Feature
"Build a SearchBar component in React.
It should have an input field and a search button.
On submit it should call the GitHub API
and fetch that username data."

---

## Prompt 4 — Profile Card UI
"Create a ProfileCard component that shows
avatar, name, username, bio, join date,
followers, following, and repo count.
Use plain CSS only, no Tailwind."

---

## Prompt 5 — Repositories Section
"Add a RepoList component that fetches
the user repositories using repos url.
Sort them by latest updated date.
Show only top 5 with name, stars, and last updated."

---

## Prompt 6 — Battle Mode
"Build a Battle component with two input fields.
Fetch both GitHub users at the same time
using Promise.all.
Compare their followers and total stars.
Show winner in green and loser in red."

---

## Prompt 7 — Dark Mode
"Add a dark and light mode toggle button.
Default should be dark mode.
Save preference in localStorage.
Use CSS variables for theming."

---

## Prompt 8 — Loading and Error States
"Add a loading spinner while API is fetching data.
Show proper error messages for 404 and network errors.
Make sure the app never crashes."

---

## Prompt 9 — CSS and Responsive Design
"Write complete custom CSS in style.css file.
Make the design clean and modern with dark theme.
Add hover effects and smooth transitions.
Make it fully responsive for mobile and desktop."

---

## Prompt 10 — Final Review
"Review all components for clean code quality.
Make sure each file has one responsibility.
Check that folder structure is properly organized."
